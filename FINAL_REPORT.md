# Mortgage IDP Workbench: Project Status Report
**Status**: Phase 2 Complete (Operational & Local-First)
**Arch**: Node.js | React | Python (FastAPI) | SQLite

## 1. System Architecture
We have successfully built a fully local Intelligent Document Processing (IDP) pipeline that removes all dependencies on cloud storage (S3) and heavy database clusters.

- **Local Storage Pipe**: All PDFs, exploded PNGs, and final exports are stored in `c:\laragon\www\doc-proj\storage`.
- **Relational DB**: SQLite (Prisma) manages all document states, page associations, and audit trails.
- **AI Service**: Python FastAPI engine running PaddleOCR and OpenCV for specialized mortgage processing.
